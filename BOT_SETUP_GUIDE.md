## Функции бота

### 1. Связывание аккаунта
- Пользователь нажимает "Подключить Telegram" в Dashboard
- Получает ссылку `t.me/EnergyLogic_Support_bot?start=token`
- Нажимает "Start" в боте
- Аккаунт автоматически связывается

### 2. Команды бота
- `/start` - Приветствие и инструкции
- `/help` - Справка по командам
- `/status` - Статус аккаунта и последние обращения
- `/purchases` - Список покупок
- `/unlink` - Отвязать аккаунт

### 3. Автоматические уведомления
- Ответы на обращения в поддержку
- Изменения статуса покупок
- Новые курсы и материалы
- Чеки и документы
- Напоминания об активности

## Тестирование бота

### 1. Проверка информации о боте
```bash

```

### 2. Тест команд
1. Найдите бота в Telegram: `@EnergyLogic_Support_bot`
2. Нажмите "Start"
3. Попробуйте команды:
   - `/help`
   - `/status` (после связывания аккаунта)
   - `/purchases` (после связывания аккаунта)

### 3. Тест связывания аккаунта
1. Войдите в Dashboard
2. Перейдите в "Обращения"
3. Нажмите "Подключить Telegram"
4. Откроется Telegram с ссылкой на бота
5. Нажмите "Start"
6. Аккаунт должен связаться

### 4. Тест уведомлений
1. Создайте обращение в Dashboard
2. Ответьте на него в админ-панели
3. Пользователь должен получить уведомление в Telegram
