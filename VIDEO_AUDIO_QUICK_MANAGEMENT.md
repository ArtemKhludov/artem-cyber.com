# 🎥🔊 Быстрое управление видео и аудио

## 🎯 Новые возможности в админ-панели

### 📊 Отображение медиа-контента
В таблице курсов теперь отображается:
- **Количество видео** для каждого курса
- **Статус аудио**: "Есть аудио" или "Нет аудио"

### ⚡ Быстрые действия

#### Для видео:
- **Кнопка ➕** - Добавить новое видео
- **Кнопка ➖** - Удалить последнее видео
- **Кнопка ➕** (синяя) - Включить видео для курса

#### Для аудио:
- **Кнопка ➕** - Включить аудио
- **Кнопка ➖** - Отключить аудио

## 🎛️ Как использовать

### 1. Управление видео

#### Включить видео для нового курса:
1. Найдите курс в таблице
2. В колонке "Файлы" найдите "Нет видео"
3. Нажмите кнопку ➕ рядом с текстом
4. Система автоматически:
   - Включит видео для курса
   - Создаст первое видео с заглушкой
   - Обновит счетчик

#### Добавить видео:
1. Найдите курс с видео
2. В колонке "Файлы" найдите "X видео"
3. Нажмите кнопку ➕ рядом с количеством
4. Система добавит новое видео с заглушкой

#### Удалить видео:
1. Найдите курс с видео
2. В колонке "Файлы" найдите "X видео"
3. Нажмите кнопку ➖ рядом с количеством
4. Система удалит последнее видео

### 2. Управление аудио

#### Включить аудио:
1. Найдите курс в таблице
2. В колонке "Файлы" найдите "Нет аудио"
3. Нажмите кнопку ➕ рядом с текстом
4. Система включит аудио с заглушкой

#### Отключить аудио:
1. Найдите курс с аудио
2. В колонке "Файлы" найдите "Есть аудио"
3. Нажмите кнопку ➖ рядом с текстом
4. Система отключит аудио

## 🎨 Визуальные индикаторы

### Цвета кнопок:
- **🟢 Зеленая ➕** - Добавить рабочую тетрадь
- **🔴 Красная ➖** - Удалить рабочую тетрадь
- **🔵 Синяя ➕** - Включить рабочие тетради
- **🟣 Фиолетовая ➕** - Добавить видео
- **🔴 Красная ➖** - Удалить видео
- **🟣 Фиолетовая ➕** - Включить видео
- **🔴 Красная ➕** - Включить аудио
- **🔴 Красная ➖** - Отключить аудио

### Состояния:
- **"X тетрадей"** - Курс с рабочими тетрадями
- **"Нет тетрадей"** - Курс без рабочих тетрадей
- **"X видео"** - Курс с видео
- **"Нет видео"** - Курс без видео
- **"Есть аудио"** - Курс с аудио
- **"Нет аудио"** - Курс без аудио

## ⚠️ Важные замечания

### После быстрого добавления:
- **Обязательно загрузите файлы** через кнопку ✏️ (редактировать курс)
- Или отредактируйте курс и загрузите файлы вручную

### Удаление медиа:
- **Видео**: удаляется последнее видео (по порядку)
- **Аудио**: полностью отключается
- Подтверждение удаления через диалог

### Автоматические обновления:
- Счетчики обновляются автоматически
- Локальное состояние обновляется без перезагрузки
- Изменения сохраняются в базе данных

## 🔧 Технические детали

### Видео:
- Хранятся в массиве `video_urls`
- Счетчик `video_count` обновляется автоматически
- Флаг `has_videos` управляет отображением

### Аудио:
- Хранится в поле `audio_url`
- Флаг `has_audio` управляет отображением
- Одиночный файл на курс

### Заглушки:
- Видео: `https://placeholder.com/video.mp4`
- Аудио: `https://placeholder.com/audio.mp3`
- Рабочие тетради: `https://placeholder.com/workbook.pdf`
