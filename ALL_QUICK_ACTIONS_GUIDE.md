# ⚡ Полное руководство по быстрым действиям

## 🎯 Обзор системы

Теперь в админ-панели доступны быстрые действия для управления всеми типами контента курсов:
- **📚 Рабочие тетради** (множественные)
- **🎥 Видео** (множественные)
- **🔊 Аудио** (одиночное)

## 📊 Отображение в таблице курсов

### Колонка "Файлы" содержит:
1. **Иконки типов контента** (обложка, PDF, превью, тетрадь, аудио, видео)
2. **Счетчики и быстрые действия** для каждого типа

## ⚡ Быстрые действия

### 📚 Рабочие тетради
```
🟢 ➕ Добавить тетрадь
🔴 ➖ Удалить последнюю тетрадь
🔵 ➕ Включить рабочие тетради
```

### 🎥 Видео
```
🟣 ➕ Добавить видео
🔴 ➖ Удалить последнее видео
🟣 ➕ Включить видео
```

### 🔊 Аудио
```
🔴 ➕ Включить аудио
🔴 ➖ Отключить аудио
```

## 🎨 Цветовая схема

| Цвет | Назначение |
|------|------------|
| 🟢 Зеленый | Добавить рабочие тетради |
| 🔵 Синий | Включить рабочие тетради |
| 🟣 Фиолетовый | Управление видео |
| 🔴 Красный | Удаление/отключение |

## 📋 Пошаговые инструкции

### 1. Включить контент для нового курса

#### Рабочие тетради:
1. Найдите "Нет тетрадей"
2. Нажмите синюю ➕
3. Система создаст первую тетрадь

#### Видео:
1. Найдите "Нет видео"
2. Нажмите фиолетовую ➕
3. Система создаст первое видео

#### Аудио:
1. Найдите "Нет аудио"
2. Нажмите красную ➕
3. Система включит аудио

### 2. Добавить контент к существующему курсу

#### Рабочие тетради:
1. Найдите "X тетрадей"
2. Нажмите зеленую ➕
3. Система добавит новую тетрадь

#### Видео:
1. Найдите "X видео"
2. Нажмите фиолетовую ➕
3. Система добавит новое видео

### 3. Удалить контент

#### Рабочие тетради:
1. Найдите "X тетрадей"
2. Нажмите красную ➖
3. Подтвердите удаление

#### Видео:
1. Найдите "X видео"
2. Нажмите красную ➖
3. Подтвердите удаление

#### Аудио:
1. Найдите "Есть аудио"
2. Нажмите красную ➖
3. Подтвердите отключение

## ⚠️ Важные замечания

### После быстрого добавления:
- **Всегда загружайте реальные файлы** через редактирование курса
- **Заглушки** используются только для создания структуры
- **Счетчики** обновляются автоматически

### Безопасность:
- **Подтверждение** требуется для всех удалений
- **Откат** невозможен после удаления
- **Резервные копии** рекомендуется делать заранее

### Производительность:
- **Локальное обновление** без перезагрузки страницы
- **Мгновенная обратная связь** через уведомления
- **Автоматическое сохранение** в базе данных
