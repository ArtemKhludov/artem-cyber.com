## 🎯 Как это работает

### Создание обращения
1. Пользователь заходит в Dashboard → "Мои покупки"
2. Нажимает "Сообщить о проблеме"
3. **НОВОЕ**: Заполняет расширенную форму:
   - Описание проблемы
   - Email (обязательно)
   - Телефон (необязательно)
   - Выбирает способы уведомлений
   - Если выбрал Telegram - указывает username
4. Отправляет обращение

### Обработка обращения
1. **Данные сохраняются** в профиле пользователя
2. **Админ получает уведомление** в Telegram (основной бот)
3. **В админ-панели** отображается полная информация о пользователе
4. **Админ отвечает** через админ-панель

### Уведомления пользователю
1. **Email уведомление** (если включено)
2. **Telegram уведомление** через EnergyLogic Support Bot (если включено)
3. **В уведомлении** есть кнопка перехода в личный кабинет
4. **Пользователь видит** ответ в Dashboard

### Дополнительные возможности бота
- **Команда `/purchases`** - показывает покупки пользователя
- **Команда `/status`** - показывает статус аккаунта и обращения
- **Автоматические уведомления** о чеках, статусах, новых курсах
- **Связывание аккаунта** через безопасные токены
