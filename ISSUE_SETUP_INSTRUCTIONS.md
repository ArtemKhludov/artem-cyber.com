# Email для отправки ответов (Resend API)

## 🚀 Как протестировать:

### 1. **Создание обращения пользователем:**
1. Откройте дашборд пользователя
2. Нажмите кнопку "Сообщить о проблеме"
3. Заполните форму (email автоматически подтянется)
4. Нажмите "Отправить"
5. Проверьте, что пришло уведомление в Telegram

### 2. **Ответ админа:**
1. Откройте админ-панель
2. Перейдите на таб "Обращения"
3. Выберите обращение
4. Напишите ответ и нажмите "Отправить"
5. Проверьте, что пользователь получил email

### 3. **Просмотр ответов пользователем:**
1. В дашборде пользователя перейдите на таб "Обращения"
2. Увидьте свой список обращений
3. Просмотрите ответы поддержки

## 🔍 Проверка работы:

### Telegram-уведомления:
- При создании обращения должно прийти сообщение в тему `CourseIssues` (ID: 8)
- Используется бот `EnergyLogicIssuesBot`

### Email-уведомления:
- При ответе админа пользователь получает email на указанный адрес
- Email содержит текст ответа и статус обращения

### База данных:
- Обращения сохраняются в таблице `issue_reports`
- Ответы сохраняются в таблице `issue_replies`
- Все действия логируются в `audit_logs`
