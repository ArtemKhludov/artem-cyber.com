# 🎓 Руководство по управлению курсами в админ-панели

## 📋 Обзор

Новая вкладка **"Курсы"** в админ-панели позволяет полностью управлять мини-курсами и PDF документами. Вы можете создавать, редактировать и удалять курсы, а также загружать все необходимые файлы.

## 🚀 Основные возможности

### ✅ Создание курса
- **Название и описание** - основная информация о курсе
- **Тип курса** - мини-курс или обычный PDF
- **Цена** - стоимость курса в рублях
- **Количество страниц** - для PDF документов

### 📁 Управление файлами
- **Обложка курса** - изображение для предпросмотра
- **Основной PDF** - главный документ курса
- **Превью видео** - короткое видео для предпросмотра
- **Рабочая тетрадь** - дополнительный PDF с упражнениями
- **Аудио-настройка** - MP3 файл для практики
- **Видео уроки** - до 3 видео уроков

### ⚙️ Дополнительные настройки
- **Продолжительность курса** - общее время в минутах
- **Количество видео** - сколько видео уроков включено
- **Чекбоксы** - какие материалы включены в курс

## 🎯 Как использовать

### 1. Создание нового курса
1. Нажмите кнопку **"Добавить курс"**
2. Заполните основную информацию (название, описание, цена)
3. Выберите тип курса (мини-курс или PDF)
4. Загрузите файлы через кнопки **📤** или введите URL вручную
5. Настройте дополнительные параметры
6. Нажмите **"Сохранить"**

### 2. Редактирование курса
1. Найдите нужный курс в таблице
2. Нажмите кнопку **✏️** (Редактировать)
3. Измените нужные поля
4. Загрузите новые файлы при необходимости
5. Нажмите **"Сохранить"**

### 3. Загрузка файлов
- **Автоматическая загрузка**: Нажмите кнопку **📤** рядом с полем
- **Ручной ввод**: Введите URL файла в поле
- **Поддерживаемые форматы**:
  - Изображения: PNG, JPG, JPEG
  - PDF: PDF
  - Видео: MP4, AVI, MOV
  - Аудио: MP3, WAV

## 📊 Таблица курсов

В таблице отображается:
- **Название** - название курса и краткое описание
- **Тип** - мини-курс или PDF с соответствующими иконками
- **Цена** - стоимость курса
- **Материалы** - иконки показывают, какие файлы загружены:
  - 🖼️ Обложка
  - 📄 PDF
  - ▶️ Превью видео
  - 📚 Рабочая тетрадь
  - 🔊 Аудио
  - 🎥 Видео уроки
- **Создан** - дата создания
- **Действия** - кнопки редактирования и удаления

## 🔧 Технические детали

### Структура файлов в Supabase Storage
```
course-materials/
└── courses/
    └── [название-курса]/
        ├── cover.png          # Обложка
        ├── main.pdf           # Основной PDF
        ├── preview.mp4        # Превью видео
        ├── workbook.pdf       # Рабочая тетрадь
        ├── audio.mp3          # Аудио-настройка
        └── videos/
            ├── video1.mp4     # Видео урок 1
            ├── video2.mp4     # Видео урок 2
            └── video3.mp4     # Видео урок 3
```

### Безопасность
- Все файлы загружаются в публичный bucket `course-materials`
- Приватные материалы (основные PDF, видео, аудио) доступны только оплатившим пользователям
- Публичные материалы (обложки, превью) доступны всем

## 🎨 Типы курсов

### Мини-курс
- Включает все компоненты: PDF, видео, аудио, рабочую тетрадь
- Отображается с превью видео на странице курса
- Кнопка "Купить мини-курс"

### PDF
- Только основной PDF документ
- Отображается с PDF превью на странице
- Кнопка "Купить PDF-файл"

## 🚨 Важные замечания

1. **Название курса** используется для создания папки в Storage
2. **Все поля обязательны** для корректной работы
3. **Файлы загружаются автоматически** при выборе через кнопку 📤
4. **URL можно вводить вручную** если файл уже загружен
5. **Удаление курса** удаляет все связанные файлы

## 🆘 Поддержка

При возникновении проблем:
1. Проверьте, что все обязательные поля заполнены
2. Убедитесь, что файлы загружены корректно
3. Проверьте права доступа к Supabase Storage
4. Обратитесь к разработчику для решения технических вопросов
