# ⚡ Быстрое управление рабочими тетрадями

## 🎯 Новые возможности в админ-панели

### 📊 Отображение количества тетрадей
В таблице курсов теперь отображается:
- **Количество рабочих тетрадей** для каждого курса
- **Статус**: "X тетрадей" или "Нет тетрадей"

### ⚡ Быстрые действия

#### Для курсов БЕЗ рабочих тетрадей:
- **Кнопка ➕** - Включить рабочие тетради и создать первую тетрадь
- Автоматически устанавливает `has_workbook = true`
- Создает "Рабочая тетрадь 1" с пустым URL

#### Для курсов С рабочими тетрадями:
- **Кнопка ➕** - Добавить новую рабочую тетрадь
- **Кнопка ➖** - Удалить последнюю рабочую тетрадь
- Автоматически обновляет счетчик `workbook_count`

## 🎛️ Как использовать

### 1. Включить рабочие тетради для нового курса
1. Найдите курс в таблице
2. В колонке "Файлы" найдите "Нет тетрадей"
3. Нажмите кнопку ➕ рядом с текстом
4. Система автоматически:
   - Включит рабочие тетради для курса
   - Создаст первую тетрадь
   - Обновит счетчик

### 2. Добавить рабочую тетрадь
1. Найдите курс с рабочими тетрадями
2. В колонке "Файлы" найдите "X тетрадей"
3. Нажмите кнопку ➕ рядом с количеством
4. Система создаст новую тетрадь с названием "Рабочая тетрадь X"

### 3. Удалить рабочую тетрадь
1. Найдите курс с рабочими тетрадями
2. В колонке "Файлы" найдите "X тетрадей"
3. Нажмите кнопку ➖ рядом с количеством
4. Система удалит последнюю тетрадь (с максимальным order_index)

## ⚠️ Важные замечания

### После быстрого добавления:
- **Обязательно загрузите файл** через кнопку 📚 (управление тетрадями)
- Или отредактируйте тетрадь и загрузите файл вручную

### Удаление тетрадей:
- Удаляется **последняя** тетрадь (по порядку)
- Если тетрадей не осталось, кнопка ➖ становится неактивной
- Подтверждение удаления через диалог

### Автоматические обновления:
- Счетчик `workbook_count` обновляется автоматически
- Локальное состояние обновляется без перезагрузки страницы
- Изменения сохраняются в базе данных

## 🎨 Визуальные индикаторы

### Цвета кнопок:
- **🟢 Зеленая ➕** - Добавить тетрадь
- **🔴 Красная ➖** - Удалить тетрадь  
- **🔵 Синяя ➕** - Включить рабочие тетради

### Состояния:
- **"X тетрадей"** - Курс с рабочими тетрадями
- **"Нет тетрадей"** - Курс без рабочих тетрадей
- **Неактивная ➖** - Нет тетрадей для удаления

## 🚀 Преимущества

- **Быстрое управление** без открытия модальных окон
- **Автоматическое именование** тетрадей
- **Мгновенное обновление** счетчиков
- **Интуитивный интерфейс** с понятными иконками
- **Безопасность** с подтверждением удаления
