# 📚 Инструкции по настройке системы множественных рабочих тетрадей

## 🚀 Быстрый старт

### 1. Выполните SQL скрипт в Supabase Dashboard

1. **Откройте Supabase Dashboard** → ваш проект → SQL Editor
2. **Скопируйте и выполните** содержимое файла `update_workbooks_system.sql`
3. **Проверьте**, что все команды выполнились без ошибок

### 2. Запустите миграцию структуры файлов

```bash
node scripts/migrate-workbooks-system.js
```

### 3. Проверьте создание структуры

- ✅ Таблица `course_workbooks` создана
- ✅ Папки `workbooks/` созданы в Storage
- ✅ Счетчики `workbook_count` обновлены

## 🎛️ Использование

### Добавление рабочих тетрадей через админ-панель

1. **Откройте админ-панель** → вкладка "Курсы"
2. **Найдите нужный курс** в таблице
3. **Нажмите кнопку 📚** (управление рабочими тетрадями)
4. **Добавьте рабочие тетради** с помощью кнопки "Добавить тетрадь"

### Структура файлов в Storage

```
course-materials/courses/[course-slug]/
├── cover.png
├── main.pdf
├── preview.mp4
├── audio.mp3
├── videos/
│   ├── video1.mp4
│   ├── video2.mp4
│   └── video3.mp4
└── workbooks/          ← Новая папка
    ├── workbook-1.pdf
    ├── workbook-2.pdf
    ├── workbook-3.pdf
    └── ...
```

## 🔧 API Endpoints

- `GET /api/admin/workbooks?documentId={id}` - получить рабочие тетради
- `POST /api/admin/workbooks` - создать рабочую тетрадь
- `PUT /api/admin/workbooks` - обновить рабочую тетрадь
- `DELETE /api/admin/workbooks?id={id}` - удалить рабочую тетрадь

## 🎨 Отображение на сайте

Система автоматически определяет тип курса:
- **Множественные рабочие тетради** - отображаются списком
- **Одиночная рабочая тетрадь** - отображается как раньше (обратная совместимость)

## 🚨 Возможные проблемы

### Ошибка "relation profiles does not exist"
- **Решение**: SQL скрипт исправлен, RLS политики упрощены

### Ошибка "exec_sql function not found"
- **Решение**: Выполните SQL скрипт вручную в Supabase Dashboard

### Файлы не отображаются
- **Проверьте**: права доступа к Storage bucket
- **Проверьте**: корректность URL файлов в базе данных
