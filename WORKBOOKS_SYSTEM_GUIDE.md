# 📚 Руководство по системе множественных рабочих тетрадей

## 🎯 Обзор

Новая система позволяет добавлять неограниченное количество рабочих тетрадей для каждого курса. Каждая рабочая тетрадь имеет свое название, описание, файл и порядок отображения.

## 🏗️ Архитектура системы

### 📊 База данных

**Новая таблица `course_workbooks`:**
```sql
- id (UUID) - уникальный идентификатор
- document_id (UUID) - ссылка на курс
- title (VARCHAR) - название рабочей тетради
- description (TEXT) - описание
- file_url (TEXT) - URL файла PDF
- order_index (INTEGER) - порядок отображения
- is_active (BOOLEAN) - активна ли тетрадь
- created_at, updated_at - временные метки
```

**Обновленная таблица `documents`:**
```sql
- workbooks (JSONB) - массив рабочих тетрадей (deprecated)
- workbook_count (INTEGER) - количество активных тетрадей
```

### 📁 Структура файлов в Storage

```
course-materials/courses/[course-slug]/
├── cover.png
├── main.pdf
├── preview.mp4
├── audio.mp3
├── videos/

└── workbooks/          ← Новая папка

    └── ...
```

## 🚀 Развертывание

### 1. Выполните SQL скрипт
```bash
# В Supabase Dashboard выполните:
update_workbooks_system.sql
```

### 2. Запустите миграцию
```bash
node scripts/migrate-workbooks-system.js
```

### 3. Проверьте создание структуры
- Таблица `course_workbooks` создана
- Папки `workbooks/` созданы в Storage
- Счетчики `workbook_count` обновлены

## 🎛️ Управление через админ-панель

### Добавление рабочих тетрадей

1. **Откройте админ-панель** → вкладка "Курсы"
2. **Найдите нужный курс** в таблице
3. **Нажмите кнопку 📚** (управление рабочими тетрадями)
4. **Нажмите "Добавить тетрадь"**
5. **Заполните форму:**
   - Название (обязательно)
   - Описание (опционально)
   - Загрузите PDF файл или введите URL
6. **Сохраните**

### Редактирование рабочих тетрадей

1. **В списке рабочих тетрадей** нажмите кнопку ✏️
2. **Измените нужные поля:**
   - Название
   - Описание
   - Файл (загрузите новый или измените URL)
3. **Сохраните изменения**

### Управление порядком

- **Кнопки ↑↓** для изменения порядка отображения
- **Автоматическая нумерация** order_index
- **Переупорядочивание** в реальном времени

### Управление видимостью

- **Кнопка "Активна/Скрыта"** для показа/скрытия тетради
- **Скрытые тетради** не отображаются на сайте
- **Счетчик workbook_count** обновляется автоматически

## 🔧 API Endpoints

### GET /api/admin/workbooks?documentId={id}
Получить все рабочие тетради для курса

### POST /api/admin/workbooks
Создать новую рабочую тетрадь
```json
{
  "document_id": "uuid",
  "title": "Название тетради",
  "description": "Описание",
  "file_url": "https://...",
  "order_index": 1
}
```

### PUT /api/admin/workbooks
Обновить рабочую тетрадь
```json
{
  "id": "uuid",
  "title": "Новое название",
  "is_active": true
}
```

### DELETE /api/admin/workbooks?id={id}
Удалить рабочую тетрадь

## 🆘 Поддержка

### Частые проблемы

**Рабочие тетради не отображаются:**
1. Проверьте `is_active = true`
2. Убедитесь, что файл загружен корректно
3. Проверьте права доступа к Storage

**Ошибка загрузки файла:**
1. Проверьте размер файла
2. Убедитесь, что формат PDF
3. Проверьте подключение к Supabase

**Порядок не сохраняется:**
1. Обновите страницу
2. Проверьте order_index в базе данных
3. Убедитесь, что триггеры работают
