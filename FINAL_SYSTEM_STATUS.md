## 🔧 ЧТО НУЖНО НАСТРОИТЬ ДЛЯ ПОЛНОЙ РАБОТЫ

### 1. 📧 **EMAIL УВЕДОМЛЕНИЯ (Resend)**
```bash
# Переменные окружения
RESEND_API_KEY=re_your_api_key_here
NOTIFY_SENDER_EMAIL=noreply@energylogic.com
```

**Шаги:**
1. Зарегистрироваться на [resend.com](https://resend.com)
2. Создать API ключ
3. Настроить домен
4. Добавить переменные в `.env.local`

### 2. 📱 **TELEGRAM УВЕДОМЛЕНИЯ**
```bash
# Переменные окружения
TELEGRAM_BOT_TOKEN=your_admin_bot_token
TELEGRAM_CHAT_ID=your_admin_chat_id
USER_TELEGRAM_BOT_TOKEN=your_user_bot_token
USER_TELEGRAM_BOT_USERNAME=your_user_bot_username
```

**Шаги:**
1. Создать ботов через @BotFather
2. Настроить webhook'и
3. Добавить переменные в `.env.local`

### 3. ⏰ **CRON JOB**
```bash
# Переменная окружения
CRON_SECRET=your_secure_random_string
```

**Варианты:**
- Vercel Cron (рекомендуется)
- Внешний сервис cron-job.org
- Настроить каждые 5 минут

## 📋 **ИНСТРУКЦИЯ ПО ЗАПУСКУ**

### **Шаг 1: Настройка переменных окружения**
```bash
# Скопируйте env.example в .env.local
cp env.example .env.local

# Добавьте ваши ключи:
RESEND_API_KEY=your_key_here
TELEGRAM_BOT_TOKEN=your_bot_token
USER_TELEGRAM_BOT_TOKEN=your_user_bot_token
```

### **Шаг 2: Настройка Resend**
1. Зарегистрируйтесь на resend.com
2. Создайте API ключ
3. Настройте домен
4. Добавьте ключ в `.env.local`

### **Шаг 3: Настройка Telegram**
1. Создайте ботов через @BotFather
2. Настройте webhook'и
3. Добавьте токены в `.env.local`

### **Шаг 4: Настройка Cron**
1. Добавьте `CRON_SECRET` в `.env.local`
2. Настройте Vercel Cron или внешний сервис

### **Шаг 5: Тестирование**
```bash
# Запустите полный тест
node test_complete_system.js
```
