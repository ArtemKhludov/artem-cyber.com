# 🎓 Руководство по развертыванию мини-курсов

Это руководство описывает процесс трансформации PDF-файлов в мини-курсы с поддержкой видео, аудио и рабочих тетрадей.

## 📋 Обзор изменений

### Что изменилось:
- **PDF-файлы** теперь могут быть **мини-курсами**
- Добавлена поддержка **видео-уроков** (2-3 видео по 5-7 минут)
- Добавлена поддержка **рабочих тетрадей** (PDF, 7-10 страниц)
- Добавлена поддержка **аудио-настроек** (MP3, 5 минут)
- Обновлен интерфейс покупки: "Купить мини-курс" вместо "Купить PDF-файл"
- Обновлена страница скачивания для доступа ко всем материалам курса

### Что сохранилось:
- Вся существующая функциональность PDF-файлов
- Процесс оплаты и покупки
- Админ-панель
- Дизайн и визуальное оформление

## 🚀 Пошаговое развертывание

### Шаг 1: Обновление базы данных

1. **Выполните SQL скрипт** для расширения схемы:
   ```bash
   # В Supabase Dashboard или через psql
   psql -h your-db-host -U your-user -d your-db -f extend_documents_for_mini_courses.sql
   ```

2. **Проверьте** что новые поля добавлены:
   ```sql
   \d documents
   ```

### Шаг 2: Настройка Supabase Storage

1. **Запустите скрипт настройки Storage**:
   ```bash
   node scripts/setup-storage-structure.js
   ```

   Этот скрипт автоматически:
   - Создаст bucket `course-materials` в Supabase Storage
   - Создаст структуру папок для мини-курсов
   - Загрузит заглушки для всех типов файлов
   - Настроит политики доступа

2. **Проверьте созданную структуру** в Supabase Dashboard:
   ```
   course-materials/
   └── courses/
       └── quantum-intention/
           ├── main.pdf (заглушка)
           ├── workbook.pdf (заглушка)
           ├── videos/
           │   ├── video1.mp4 (заглушка)
           │   ├── video2.mp4 (заглушка)
           │   ├── video3.mp4 (заглушка)
           │   └── preview.mp4 (заглушка)
           └── audio.mp3 (заглушка)
   ```

### Шаг 3: Миграция существующих документов

1. **Запустите миграцию** (скрипт автоматически использует заглушки):
   ```bash
   node scripts/migrate-to-mini-courses.js
   ```

2. **Проверьте результат**:
   ```sql
   SELECT title, course_type, has_videos, has_workbook, has_audio 
   FROM documents;
   ```

3. **Замените заглушки реальными файлами** через админ-панель:
   - Откройте админ-панель
   - Найдите документ "Квантовая Архитектура Намерения"
   - Нажмите "Редактировать"
   - Загрузите реальные файлы через кнопки загрузки

### Шаг 4: Тестирование

1. **Проверьте страницу PDF**:
   - Откройте `/pdf/[id]` для мини-курса
   - Убедитесь, что отображается контент мини-курса
   - Проверьте кнопку "Купить мини-курс"

2. **Проверьте процесс покупки**:
   - Пройдите полный цикл покупки
   - Убедитесь, что в личном кабинете доступны все материалы

3. **Проверьте админ-панель**:
   - Откройте админ-панель
   - Убедитесь, что можно редактировать мини-курсы

## 🎬 Создание контента для мини-курса

### Структура мини-курса "Квантовая Архитектура Намерения":

#### 1. Основной PDF (20-30 страниц)
- **Содержание**: Плотный текст + схемы
- **Формат**: PDF
- **Размещение**: `public/courses/quantum-intention/main.pdf`

#### 2. Рабочая тетрадь (7-10 страниц)
- **Содержание**:
  - Таблица "Истинное намерение или ложное?"
  - Чек-лист "Алгоритм перезаписи в 5 шагов"
  - Шаблон "Дневник намерения (на неделю)"
- **Формат**: PDF
- **Размещение**: `public/courses/quantum-intention/workbook.pdf`

#### 3. Видео-уроки (3 видео по 5-7 минут)
- **Видео 1**: "Что такое намерение до мысли"
- **Видео 2**: "Ошибки при работе с намерением"
- **Видео 3**: "Пример перезаписи на простом кейсе"
- **Формат**: MP4
- **Размещение**: `public/courses/quantum-intention/videos/`

#### 4. Аудио-настройка (5 минут)
- **Содержание**: "Закрой глаза, почувствуй искру, зафиксируй её, отпусти шум"
- **Формат**: MP3
- **Размещение**: `public/courses/quantum-intention/audio.mp3`

#### 5. Превью видео
- **Назначение**: Отображение на странице курса
- **Формат**: MP4
- **Размещение**: `public/courses/quantum-intention/preview.mp4`

## 🔧 Настройка админ-панели

### Добавление компонента в админ-панель:

1. **Импортируйте компонент** в админ-страницу:
   ```tsx
   import { MiniCourseManagement } from '@/components/admin/MiniCourseManagement'
   ```

2. **Добавьте вкладку** для управления мини-курсами:
   ```tsx
   <TabsContent value="mini-courses">
     <MiniCourseManagement />
   </TabsContent>
   ```

### Управление файлами через админ-панель:

1. **Загрузка файлов**:
   - Нажмите кнопку "Редактировать" для документа
   - Используйте кнопки загрузки (📤) рядом с каждым полем
   - Файлы автоматически загружаются в Supabase Storage
   - URL автоматически обновляется в форме

2. **Типы файлов**:
   - **Основной PDF**: Загружается в `courses/[course-slug]/main.pdf`
   - **Рабочая тетрадь**: Загружается в `courses/[course-slug]/workbook.pdf`
   - **Видео**: Загружаются в `courses/[course-slug]/videos/video1.mp4`, `video2.mp4`, `video3.mp4`
   - **Превью видео**: Загружается в `courses/[course-slug]/videos/preview.mp4`
   - **Аудио**: Загружается в `courses/[course-slug]/audio.mp3`

3. **Автоматическое именование**:
   - Название курса автоматически преобразуется в slug (например: "Квантовая Архитектура Намерения" → "quantum-intention")
   - Файлы сохраняются в соответствующую папку

- Все медиа-файлы должны быть доступны только после покупки
- Используйте signed URLs для временного доступа
- Не храните чувствительные данные в публичных папках

### Совместимость:

- Существующие PDF продолжают работать
- Можно постепенно мигрировать документы
