# Как загрузить новый фавикон

## Способ 1: Прямая замена файла (Самый простой) ⭐

### Шаг 1: Подготовьте файл
1. У вас должна быть картинка (PNG, JPG, SVG) или готовый `favicon.ico`
2. Если у вас PNG/JPG → конвертируйте в ICO:
   - Откройте https://favicon.io/favicon-converter/
   - Загрузите вашу картинку
   - Скачайте готовый `favicon.ico`

### Шаг 2: Замените файл в проекте

**Вариант A: Через Finder (Mac) или Проводник (Windows)**

1. Откройте Finder (Mac) или Проводник (Windows)
2. Перейдите в папку проекта:
   ```
   /Users/artem/Projects/work/energylogic-site/app/
   ```
3. Найдите файл `favicon.ico`
4. **Перетащите** ваш новый `favicon.ico` файл и замените старый
   - Или скопируйте новый файл, удалите старый, переименуйте новый в `favicon.ico`

**Вариант B: Через командную строку**

```bash
# Перейдите в папку проекта
cd /Users/artem/Projects/work/energylogic-site

# Замените файл (положите новый favicon.ico в папку app/)
# Или используйте команду cp:
cp /путь/к/вашему/новому/favicon.ico app/favicon.ico
```

### Шаг 3: Проверьте результат
1. Перезапустите dev сервер:
   ```bash
   npm run dev
   ```
2. Откройте браузер: http://localhost:3000
3. Очистите кеш: `Cmd+Shift+R` (Mac) или `Ctrl+Shift+R` (Windows)
4. Проверьте вкладку браузера - должен появиться новый фавикон

## Способ 2: Drag & Drop в Cursor (если поддерживается)

1. Откройте папку `app` в Cursor
2. Попробуйте перетащить файл `favicon.ico` прямо в список файлов
3. Если появится диалог "Replace?" - нажмите "Yes"

## Способ 3: Через онлайн-конвертер и загрузку

### Если у вас есть PNG/JPG изображение:

1. **Конвертируйте онлайн:**
   - https://favicon.io/favicon-converter/
   - https://convertio.co/png-ico/
   - https://realfavicongenerator.net/

2. **Скачайте готовый `favicon.ico`**

3. **Загрузите в проект:**
   - Откройте Finder/Проводник
   - Перейдите в `app/` папку проекта
   - Перетащите новый файл `favicon.ico`
   - Подтвердите замену

---

## Способ 4: Использование icon.png (Современный подход)

Вместо `favicon.ico` можно использовать PNG:

1. **Создайте `icon.png`** размером 512x512 пикселей
2. **Положите файл в папку `app/`**:
   ```
   app/icon.png
   ```
3. Next.js автоматически подхватит его!

Не нужно конвертировать в ICO - просто PNG файл.

## Проверка после загрузки

После замены файла:
1. ✅ Файл должен быть в `app/favicon.ico`
2. ✅ Перезапустите сервер: `npm run dev`
3. ✅ Очистите кеш браузера
4. ✅ Откройте сайт - фавикон должен обновиться

**Если не видно изменений:**
- Откройте в режиме инкогнито
- Очистите кеш браузера полностью
- Подождите 1-2 минуты (браузер кеширует фавиконы)
