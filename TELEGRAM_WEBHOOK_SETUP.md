# 🤖 Настройка Telegram Webhook для пользовательского бота

## 🎯 Цель
Настроить webhook для пользовательского Telegram бота, чтобы пользователи могли получать уведомления о своих заявках.

## 📋 Пошаговая инструкция

### Шаг 1: Получение токена бота
1. Найдите бота @BotFather в Telegram
2. Отправьте команду `/newbot` или используйте существующего бота
3. Следуйте инструкциям для создания бота
4. Сохраните полученный токен

### Шаг 2: Настройка переменных окружения
Добавьте в Vercel Environment Variables:

```env
USER_TELEGRAM_BOT_TOKEN=your_bot_token_here
```

### Шаг 3: Установка webhook
Выполните команду для установки webhook:

```bash
curl -X POST "https://api.telegram.org/bot<YOUR_BOT_TOKEN>/setWebhook" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://www.energylogic-ai.com/api/telegram/webhook",
    "allowed_updates": ["message", "callback_query"]
  }'
```

Замените `<YOUR_BOT_TOKEN>` на ваш токен бота.

### Шаг 4: Проверка webhook
Проверьте, что webhook установлен:

```bash
curl "https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getWebhookInfo"
```

Ожидаемый ответ:
```json
{
  "ok": true,
  "result": {
    "url": "https://www.energylogic-ai.com/api/telegram/webhook",
    "has_custom_certificate": false,
    "pending_update_count": 0,
    "allowed_updates": ["message", "callback_query"]
  }
}
```

### Шаг 5: Тестирование бота
1. Найдите вашего бота в Telegram
2. Отправьте команду `/start`
3. Проверьте, что бот отвечает

## 🚨 Устранение проблем

### Проблема: Webhook не работает
**Решение:**
1. Проверьте правильность URL
2. Убедитесь, что сервер доступен
3. Проверьте логи в Vercel

### Проблема: Бот не отвечает
**Решение:**
1. Проверьте токен бота
2. Убедитесь, что webhook установлен
3. Проверьте переменные окружения

### Проблема: Уведомления не приходят
**Решение:**
1. Проверьте подключение пользователя к боту
2. Убедитесь, что `telegram_chat_id` сохранен в базе
3. Проверьте логи отправки уведомлений
